Source code licensing

built in the open.

NanthAI Edge is source-available. The source code is free to use for personal self-hosting and learning. Businesses and organisations that want to deploy, rebrand, or build on the codebase need a Commercial License.

Looking to use NanthAI Edge as a product?

You don't need a source code license to use NanthAI Edge. Just sign up, connect your OpenRouter key, and go. Pro features are unlocked with a one-time purchase. This page is specifically about licensing the source code for self-hosting or commercial use. Features

1. Default license

The NanthAI Edge source code is available under a source-available license that permits personal use, self-hosting, evaluation, and learning — but not commercial use. The full license text is in the LICENSE file in the repository.

2. What counts as commercial use?

Commercial Use means deploying, modifying, or building on the source code in a business context. This includes:

  • Deploying a self-hosted instance for your company or team
  • Rebranding or white-labelling the codebase as your own product
  • Using the source code as part of a product, service, or internal tooling
  • Offering a hosted version to clients or customers
  • Any use by employees or contractors on behalf of a commercial entity

3. What's free?

The following uses do NOT require a Commercial License:

  • Personal self-hosting for your own individual use
  • Academic research or student projects
  • Non-profit use for non-revenue activities
  • Evaluation or proof-of-concept work (up to 30 days)

4. What does a commercial license grant?

A Commercial License is negotiated per customer and may include:

  • Internal business use — deploy for your team or organisation
  • Rebranding and white-labelling — customise the UI, name, and branding
  • Modification and extension — build custom features on the platform
  • Organisation-wide deployment — no per-seat limits
  • Priority support — direct access for technical guidance
  • Update entitlements — access to new versions for a defined period

5. Pricing

Commercial Licenses are available as:

  • One-time purchase — perpetual license for a specific version.
  • Annual subscription — includes ongoing updates and support.

6. Frequently asked questions

Can I try it before buying a license?

Yes. The default license permits evaluation and proof-of-concept use for up to 30 days. If you need more time, let us know.

Can I modify the code?

For personal use: yes, freely. For commercial use: yes, with a Commercial License. Modifications remain subject to the license terms.

Can I redistribute my modified version?

Not without a Commercial License that explicitly permits redistribution. The default license does not allow redistribution.

Is there a free tier for startups?

Contact us. We're open to discounted or deferred licensing for early-stage startups on a case-by-case basis.

7. Get a commercial license

For pricing, inquiries, or questions about licensing, contact us at support@nanthai.tech.
